Demande de Reçu d'Impôt

Remplissez ce formulaire pour recevoir votre reçu officiel pour fins d'impôt

Organisme de bienfaisance : 770133734 RR 0001

À propos de ce formulaire

  • Ce formulaire permet de demander un reçu d'impôt pour votre donation déjà effectuée
  • Un reçu officiel vous sera envoyé par courriel dans un délai de 7 à 30 jours
  • Le reçu est valide pour votre déclaration d'impôt au Canada
  • Conservez ce reçu pour vos dossiers fiscaux
Important : Ce formulaire est uniquement pour demander un reçu d'impôt suite à une donation déjà effectuée. Pour faire un nouveau don, veuillez utiliser les autres moyens de paiement disponibles.
